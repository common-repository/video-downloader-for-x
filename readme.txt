Integrate download buttons next to embedded tweets and X posts, enabling users to download videos in MP4 and MP3 format.

== Description ==

The Video Downloader for X plugin enhances your websites features by adding a "Download" button below each embedded tweet. Your visitors can now download videos directly from Twitter posts displayed on your site by connecting to the official [X Downloader](https://x-downloader.com) API. The installation is simple, and there's no configuration neededâ€”just activate the plugin, and the download buttons will automatically appear.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/video-downloader-for-x` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin will now automatically show a "Get Video" and "Get MP3" button below each embedded Tweet in your posts.

== Frequently Asked Questions ==

= Can I download any Twitter video? =

You can download videos from public tweets. Videos from private accounts cannot be downloaded.

= Do I need a Twitter account to use this plugin? =

No, you don't need a Twitter account to download public Twitter videos.
